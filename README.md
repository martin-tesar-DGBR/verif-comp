## Build Instructions
In the terminal, run `$ mvn package` to build the project.

On Linux, run `$ java -cp lib/com.microsoft.z3.jar:target/verif-comp-1.0.jar smt.Main` to run the executable jar.

On Windows, run `$ java -cp "lib/com.microsoft.z3.jar;target/verif-comp-1.0.jar" smt.Main` to run the executable jar.

Update the build instructions if any external libraries are added.
